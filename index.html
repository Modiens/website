Ticketsystem

Ein System, welches dir beim Supporten von Discord Usern hilft! Deine Ticket Supporter können dann archiven, reopenen und das Ticket schließen..

Warn System

Dein ganz eigenes Verwarnungs System für deinen Server! Server Admins können Discord User verwarnen, entwarnen, verwarnungsregeln aufstellen und mehr..
